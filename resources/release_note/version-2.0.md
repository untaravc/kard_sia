# Release Notes v2.0

## Overview
Version 2.0 introduces a refreshed experience for lecture and student users, expands authentication options, and adds reliable notification and messaging workflows to keep everyone informed.

## What’s New
- **New UI for lecture and student**: Redesigned dashboards and navigation for clearer workflows, faster access to key actions, and a more consistent visual system.
- **Expanded authentication**: Users can now sign in with email + password, passwordless email link, phone-based login, and SSO.
- **Notifications**: In-app notifications are now available to surface important updates and reminders.
- **Email reports and WhatsApp reminders**: Automated reporting and scheduled reminders help teams stay aligned without manual follow‑ups.

## Why it matters
These updates reduce friction in daily workflows, improve access and security, and ensure timely communication across the platform.

## Notes
If you have feedback or need rollout support, please share it with the product team.
